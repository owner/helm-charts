# Claude Code Guidelines for helm-charts

## Repository Configuration

**PR Target**: All PRs should target `owner/helm-charts`, NOT `bjw-s-labs/helm-charts`.

- `origin` (https://github.com/owner/helm-charts.git) - Use this for all PRs
- `upstream` (https://github.com/bjw-s-labs/helm-charts.git) - Reference only

When creating or reviewing PRs, ensure they are opened against `owner/helm-charts`.
